## Prerequisities

- DockerDesktop
- skaffold
- ingress-nginx
- npm

## To build the project

Go to app1 folder and run "npm install".

Do the same for app2 folder.

Run "skaffold dev".

Go to kafkatest.com/get-users on your browser to test if it works.
